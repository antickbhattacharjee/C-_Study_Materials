# 🎓 C++ Coaching Repository

Welcome to the C++ Coaching Repository! This repository is designed as a structured, day-by-day learning journey to master C++ from the ground up. Whether you are a beginner or looking to solidify your object-oriented programming concepts, this syllabus has been crafted to provide both theoretical knowledge and hands-on practice.

## 🗂️ Repository Structure

The course is divided into a 10-day syllabus. Each day focuses on a specific core concept of C++ programming.

- **[Day 1: Variables, Operators & Datatypes](./Day_1_Variables_Operators_Datatypes)**
- **[Day 2: Conditional Statements](./Day_2_Conditional_Statements)**
- **[Day 3: Loops](./Day_3_Loops)**
- **[Day 4: Arrays](./Day_4_Array)**
- **[Day 5: Strings](./Day_5_String)**
- **[Day 6: Functions](./Day_6_Function)**
- **[Day 7: Structures](./Day_7_Structure)**
- **[Day 8: Object-Oriented Programming (OOP) Theory](./Day_8_OOP_Theory)**
- **[Day 9: File Handling](./Day_9_File_Handling)**
- **[Day 10: Rest & Projects](./Day_10_Rest)**

## 🚀 How to Use This Repository

Inside each day's folder, you will find:
1. **`README.md`**: Contains the theoretical definitions, syntax examples, and important differences you need to know for that day's topic.
2. **`Learning/`**: Contains the instructional C++ code examples that demonstrate the concepts.
3. **`Assignments/`**: Contains practice problems for you to solve and test your understanding.

## 🛠️ Getting Started

To run the code, you will need a C++ compiler installed on your machine (like GCC/G++ or MSVC).
You can compile any of the `.cpp` files using a terminal. For example:
```bash
g++ Day_1_Variables_Operators_Datatypes/Learning/variable.cpp -o variable
./variable
```

Happy Coding! 🎉
